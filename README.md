# Ant Design Production Template

Deploying web apps correctly is hard. Use a template.

## Tech Stack

1. Ant Design
2. TypeScript (.ts, .tsx)
3. Markdown in JSX (.md, .mdx, .js, .jsx)
4. MongoDB server side
5. GraphQL client
6. Next.JS
7. Serverless deploy to Now.sh by ZEIT

## How to setup

git clone <url>
cd <project name>
yarn
yarn dev
